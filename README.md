# boobcraft (working title!!)

Simple 3D engine/framework written in C + OpenGL for retro singleplayer games.

|------------|----------------------------------------|
| Resolution | 400x240                                |
| Framerate  | 30fps                                  |

## Installation

```
brew install sdl2
brew install glew
git clone https://github.com/dairycultist/boobcraft/
cd boobcraft
make run
```
